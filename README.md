# 📝 Task Manager Web App

A simple and interactive Todo App built using JavaScript, HTML, and CSS.

## 🚀 Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Data stored using localStorage (persistent even after refresh)
- Clean and minimal UI
- Supports Enter key for quick task addition

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Browser LocalStorage
